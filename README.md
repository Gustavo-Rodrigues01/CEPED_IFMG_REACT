## 📂 Estrutura do repositório
O repositorio é organizado em três pastas principais:

* ### 📂 Atividades
  É aqui onde ficam salvas as atividades passadas pelo professor ao longo dos modulos. Esta pasta é oganizada em modulos e aulas,
  onde, cada aula contem as atividades correspondentes ao que foi passado em sala.
  
* ### 📂 Aulas
  É aqui onde ficam salvos os exemplos e arquivos desenvolvidos durante as aulas. Esta pagina contem as aulas tanto presenciais quanto      remotas

* ### 📂 Gerenciador de tarefas
  É aqui onde foi salvo cada alteração do projeto de aplicação react durante o curso

## 📦 Como executar

Para rodar as atividades, exemplos ou projetos react deste repositorio localmente, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/](https://github.com/)[seu-usuario]/[seu-repositorio].git

2. **Acesse a pasta da aplicação:**
   ```bash
   cd nome-do-projeto
   
3. **Instale as dependências:**
   ```bash
   npm install

4. **Execute o projeto:**
   ```bash
   npm run dev
ou 
   ```bash
    npm build
    npm run preview
